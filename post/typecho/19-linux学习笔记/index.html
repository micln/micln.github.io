以下所有命令基于 ubuntu12.04 系统。centos、rhel 慎重

系统环境

grub修复

linux改为主引导

之前装mac时把引导毁掉了，pe只能修复win引导。linux引导还得自己再修复。

liveCD（我直接用的烧在U盘的ubuntu试用）。

# fdisk -l  
# mount /dev/sdaX /mnt  
# mount /dev/sdaY /mnt/boot                     #如果有单独boot分区  
# grub-install --root-directory=/mnt /dev/sda  
# init 6

重启之后就ok了～

过程中出现了warning 。不用管，重启没有问题。

grub添加到windows下

主要目的是兼顾mac

在装linux 的时候，可以把grub安装在linux磁盘/sdaX，然后在win下用easyBCD把grub添加到win下，这样在一个引导下就可以选择三个系统。

添加ssh免密码登陆

网上教程一大堆，坑在了权限

sudo chmod 600 ~/.ssh/id_rsa.pub
